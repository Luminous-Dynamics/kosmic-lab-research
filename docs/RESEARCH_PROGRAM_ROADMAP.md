## The Central Question: K-Index ↔ O/R Index Relationship

### Hypothesis

**K-Index** (consciousness coherence) and **O/R Index** (behavioral consistency) may measure different aspects of the same underlying phenomenon:

```
K-Index = Internal coherence (within agent)
O/R Index = External predictability (between agents)
```

If true: `K-Index → O/R Index → Coordination Success`

### Evidence Suggesting Connection

1. **G8 Achievement**: CMA-ES reached K=1.4202 using evolutionary (not gradient) optimization
2. **O/R Insight**: Consistent behavior aids coordination (r=-0.70***)
3. **Shared Theme**: Both reward "coherent" patterns over "random" ones

### Proposed Study: Track L (K-O/R Bridge)

**Design**:
1. Generate agents at varying K-Index levels (0.8-1.5 range)
2. Measure O/R Index for each agent in multi-agent scenarios
3. Compute correlation between K-Index and O/R Index
4. Test mediation: K → O/R → Coordination

**Hypothesis**: r > 0.5 between K-Index and O/R Index (moderate-strong correlation)
