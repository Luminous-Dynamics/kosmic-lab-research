## 💡 Key Insights from Data Acquisition

### What Worked Well
1. **World Bank API**: Excellent programmatic access, comprehensive coverage
2. **Direct CSV downloads**: Reliable for UCDP, COW, QoG
3. **Multiple sources per harmony**: Enables cross-validation and robustness
4. **Automated verification**: Immediate testing ensured data quality

### Challenges Encountered
1. **Broken links**: Several academic sites had outdated URLs (WID, Freedom House)
2. **API limits**: Some datasets require registration (V-Dem required manual)
3. **Format diversity**: Mix of CSV, XLS, XLSX, JSON required flexible parsing
4. **Encoding issues**: COW data required latin1 encoding

### Solutions Implemented
1. **Multiple sources**: Tried 2-3 URLs for each dataset
2. **Alternative indicators**: Used World Bank when specialized sources failed
3. **Flexible parsers**: Added encoding detection and multiple engine support
4. **Comprehensive documentation**: Recorded all sources and alternatives
