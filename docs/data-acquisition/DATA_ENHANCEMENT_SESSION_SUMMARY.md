## 🔬 Methodological Insights

### What Works
1. **Official APIs**: World Bank, Eurostat, UNESCO have reliable programmatic access
2. **GitHub repositories**: Raw data files from academic projects (Barro-Lee)
3. **Institutional repositories**: Harvard Dataverse, official UN sites
4. **Direct downloads**: Freedom House, GEM, UNCTAD publish open Excel/CSV

### What Doesn't Work
1. **OECD.Stat**: Requires interactive query building
2. **WIPO direct**: Moved to interactive explorer
3. **Old academic URLs**: Many 404 errors (WID, SWIID old URLs)
4. **Registration required**: LIS, some OECD databases

### Best Practices Discovered
1. **Use web search** to find current URLs when direct downloads fail
2. **Try GitHub** for academic datasets - often has raw data
3. **Check Harvard Dataverse** - many datasets archived there
4. **Prioritize UN family** (UNDP, UNESCO, UNCTAD) - stable URLs
5. **Accept alternatives** - WB Gini as good as WID for most purposes
