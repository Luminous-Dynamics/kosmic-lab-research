## Critical Improvements Implemented ✅

### 1. H7 Construction Clarified (Line 101)
**Problem**: "Black box" - unclear how HYDE data translated into H7 components
**Solution**: Added explicit functional forms and weight rationale
- Component (i): Tech sophistication = urban population share
- Component (ii): Cognitive complexity = log(total population)
- Component (iii): Institutional evolution = cropland fraction
- Weights: 40/30/30 chosen to maximize in-sample R² against modern indicators
- Clarified R² = 0.999 is calibration, not independent validation

**Impact**: Eliminates "just tuned until it fits" criticism

---

### 2. External Validation Language Tightened (Line 263)
**Problem**: Oversold correlations despite tiny sample sizes (n ≤ 6)
**Solution**: Statistical humility throughout
- "strong correlations" → "large-magnitude but under-powered correlations"
- Explicit statement: p-values "not meaningful for conventional significance testing"
- Framed as "qualitative consistency checks" not "rigorous validation"

**Impact**: Pre-empts methodologist pushback, shows sophistication

---

### 3. Bootstrap Limitations Explicit (Line 283)
**Problem**: Bootstrap CI appeared comprehensive but doesn't capture measurement error
**Solution**: Added "Important caveat" paragraph
- Bootstrap captures "internal sampling variability from proxy selection" only
- Does NOT capture measurement error in underlying datasets (V-Dem, HYDE, Seshat)
- Does NOT capture model specification uncertainty
- "True epistemic uncertainty likely larger than reported CI"

**Impact**: Honest uncertainty quantification

---

### 4. "Historical vs Aspirational Scales" Subsection (Lines 163-181) ✨ NEW
**Problem**: K(2020) = 0.91 feels "too high" - suggests 91% to ideal
**Solution**: Complete new subsection with three-part framework
- (1) **Historical ceiling vs theoretical potential**: K(t) measures observed max, not ideal
- (2) **Capacity vs actualization gap**: Formal G(t) = K_capacity - K_actualization definition
  - Capacity harmonies: H2, H4, H7 (infrastructure, innovation, progression)
  - Actualization harmonies: H1, H3, H5, H6 (governance, reciprocity, wisdom, flourishing)
- (3) **Future aspirational index**: K★(t) framework for normative targets

**Impact**: Transforms scale interpretation from ambiguous to precise
