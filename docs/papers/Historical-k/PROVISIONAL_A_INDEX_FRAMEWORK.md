## Framework: K(t) vs A(t)

### Core Distinction

| Dimension | Definition | What It Measures | Example |
|-----------|------------|------------------|---------|
| **K(t)** | Coordination Capacity | Infrastructure exists | "We have the internet (H2), universities (H5), trade networks (H3)" |
| **A(t)** | Coordination Actualization | Infrastructure used effectively | "We use networks for wisdom, education creates critical thinkers, trade builds genuine cooperation" |
| **Gap(t)** | K(t) - A(t) | Utilization deficit | "High tech capacity, low quality outcomes" |
| **Ratio(t)** | A(t) / K(t) | Efficiency of actualization | "What % of capacity translates to quality?" |

### Mathematical Framework

```
K(t) = geometric_mean(H₁, H₂, H₃, H₄, H₅, H₆, H₇)  ← Capacity
A(t) = geometric_mean(A₁, A₂, A₃, A₄, A₅, A₆, A₇)  ← Quality

where A_i = f(H_i, outcomes_i)

Gap(t) = K(t) - A(t)
Ratio(t) = A(t) / K(t)
```

## Provisional A(t) Calculation

### Expected 2020 Values

| Harmony | K₂₀₂₀ (Capacity) | A₂₀₂₀ (Quality) | Gap | Gap % |
|---------|------------------|-----------------|-----|-------|
| H1 Governance | 0.825 | 0.50-0.55 | 0.27-0.33 | 33-40% |
| H2 Interconnection | 0.871 | 0.48-0.52 | 0.35-0.39 | 40-45% |
| H3 Reciprocity | 0.720 | 0.35-0.42 | 0.30-0.37 | 42-51% |
| H4 Complexity | 0.796 | 0.52-0.58 | 0.22-0.28 | 28-35% |
| H5 Knowledge | 0.645 | 0.50-0.55 | 0.10-0.15 | 15-23% |
| H6 Wellbeing | 0.778 | 0.56-0.62 | 0.16-0.22 | 21-28% |
| H7 Technology | 0.917 | 0.50-0.55 | 0.37-0.42 | 40-46% |

**Aggregated (Geometric Mean)**:
```
K₂₀₂₀ = 0.789 (from actual computation)
A₂₀₂₀ = geometric_mean([0.50-0.55, 0.48-0.52, ...]) ≈ 0.48-0.53

Gap₂₀₂₀ = K₂₀₂₀ - A₂₀₂₀ ≈ 0.26-0.31
Ratio₂₀₂₀ = A₂₀₂₀ / K₂₀₂₀ ≈ 61-67%
```

**Key Finding**: Only ~60-65% of coordination capacity translates to coordination quality.

## Expected Results & Validation

### Expected Findings

1. **Widening Gap**: Gap(t) grows from ~0.01 (1810) to ~0.30 (2020)
2. **Declining Ratio**: Ratio(t) declines from ~75% (1900) to ~64% (2020)
3. **Largest Gaps** (2020):
   - H7 Technology: 40-46% (highest capacity, poorest use)
   - H2 Interconnection: 40-45% (networks exist, quality low)
   - H3 Reciprocity: 42-51% (trade high, trust collapsed)
4. **Smallest Gap** (2020):
   - H5 Knowledge: 15-23% (education most effective at translation)

### External Validators

**If A(t) is valid**, expect:
- **High correlation with subjective wellbeing** (r > 0.7 with World Happiness)
- **Negative correlation with crisis frequency** (r < -0.5 with conflict/crash data)
- **Lower A(t) in polarized societies** (US 2020 vs Nordic 2020)
- **K-A gap predicts governance failure** (Arab Spring, Trump, Brexit)
