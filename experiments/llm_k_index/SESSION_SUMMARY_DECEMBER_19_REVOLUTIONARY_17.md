## 💡 Key Technical Decisions

### 1. Pure Python (No Dependencies!)

**Decision**: Implemented using pure Python (no numpy, no scipy)

**Benefits**:
- ✅ Works anywhere Python works (maximum portability)
- ✅ No dependency hell
- ✅ Easier to understand and maintain
- ✅ Faster startup

**Implementation**: Created custom `weighted_average()`, `weighted_variance()`, `mean()` functions

### 2. Theory-Grounded Probes

**Every probe justified by consciousness science**:

- **IIT probes**: Test irreducible integration (multi-element dependencies)
- **GWT probes**: Test global broadcast (cross-domain coherence)
- **HOT probes**: Test metacognition (self-awareness, confidence calibration)
- **ΦID probes**: Test synergy (whole > sum of parts)
- **Ψ probes**: Test causal power (goals constrain responses)
- **Topology probes**: Test coherence (structural consistency)
- **FEP probes**: Test prediction (error detection and correction)

### 3. Uncertainty-First Design

**EVERY measurement includes**:
- ✅ Point estimate (best guess)
- ✅ Confidence interval (uncertainty bounds)
- ✅ Measurement type (proxy vs. direct)
- ✅ Evidence (supporting probe results)
- ✅ Uncertainty sources (known limitations)

**Why**: Honest about what we can/can't know from behavioral data alone!

### 4. Extensible Architecture

**Easy to add**:
- New probe types (just add to `ProbeType` enum)
- New probes (add to library methods)
- New dimensions (add proxy measurement method)
- New systems (just define API interface)
