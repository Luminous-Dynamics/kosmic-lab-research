## The Revolutionary Proposal: Phenomenology Mapping

### Core Insight

While we cannot directly measure phenomenal experience, we can:

1. **Collect extensive phenomenological reports** from humans across consciousness states
2. **Map functional measurements to reported experiences**
3. **Create "phenomenological profiles"** for different (k, k_meta, b) coordinates
4. **Use structure in qualia space** to make principled inferences

**Not claiming**: Functional properties = phenomenal properties
**Claiming**: Systematic relationships exist between functional and phenomenal that we can map

### The Framework

**Phenomenology Space**: P = {perception, affect, cognition, volition, unity, boundary, temporality}

Each dimension has characteristic qualities:

**Perception (P₁)**:
- Clarity: [dim, vivid, hyperreal]
- Richness: [sparse, moderate, overwhelming]
- Stability: [fragmentary, stable, rigid]

**Affect (P₂)**:
- Valence: [negative, neutral, positive]
- Intensity: [flat, moderate, extreme]
- Differentiation: [undifferentiated, nuanced, hyper-differentiated]

**Cognition (P₃)**:
- Coherence: [confused, clear, hyper-logical]
- Speed: [sluggish, normal, racing]
- Abstractness: [concrete, balanced, abstract]

**Volition (P₄)**:
- Agency: [absent, present, hyper-agency]
- Control: [no control, normal, complete control]
- Intentionality: [reactive, intentional, predetermined]

**Unity (P₅)**:
- Integration: [fragmented, unified, dissolution]
- Boundaries: [rigid, flexible, dissolved]
- Self-other: [merged, distinct, isolated]

**Boundary (P₆)**:
- Self-world: [merged, boundaried, isolated]
- Present-past-future: [fused, distinct, disconnected]
- Internal-external: [confused, clear, rigid]

**Temporality (P₇)**:
- Flow: [frozen, flowing, racing]
- Continuity: [disjointed, continuous, eternal]
- Nowness: [absent, present-focused, timeless]

## Implementation: The Phenomenology Mapper

### Architecture

```python
class PhenomenologyMapper:
    """Maps functional consciousness measurements to phenomenological profiles."""

    def __init__(self):
        self.dimensions = {
            "perception": ["clarity", "richness", "stability"],
            "affect": ["valence", "intensity", "differentiation"],
            "cognition": ["coherence", "speed", "abstractness"],
            "volition": ["agency", "control", "intentionality"],
            "unity": ["integration", "boundaries", "self_other"],
            "boundary": ["self_world", "present_past_future", "internal_external"],
            "temporality": ["flow", "continuity", "nowness"]
        }

        # Learned mapping functions (one per phenomenological dimension)
        self.mappers = {}
        self.training_data = []

    def add_training_sample(self, functional_props: dict, phenomenology: dict,
                           source: str, confidence: float):
        """
        Add phenomenological report with functional measurements.

        Args:
            functional_props: {k, k_meta, b, Φ, W, A, R, ...}
            phenomenology: Nested dict of phenomenological ratings
            source: "sleep", "psychedelic", "meditation", "disorder", etc.
            confidence: How reliable is this report? [0, 1]
        """
        self.training_data.append({
            "functional": functional_props,
            "phenomenology": phenomenology,
            "source": source,
            "confidence": confidence,
            "timestamp": datetime.now()
        })

    def train(self):
        """Learn mapping from functional to phenomenological."""
        # For each phenomenological dimension
        for category, subdimensions in self.dimensions.items():
            for subdim in subdimensions:
                # Extract training data
                X = []  # Functional properties
                y = []  # Phenomenological ratings
                w = []  # Confidence weights

                for sample in self.training_data:
                    X.append(self._extract_features(sample["functional"]))
                    y.append(sample["phenomenology"][category][subdim])
                    w.append(sample["confidence"])

                # Fit weighted regression (or neural network)
                mapper = self._fit_mapper(X, y, w)
                self.mappers[f"{category}.{subdim}"] = mapper

    def predict(self, functional_props: dict) -> dict:
        """
        Predict phenomenological profile from functional measurements.

        Returns:
            Nested dict of predicted phenomenological ratings
        """
        phenomenology = {}

        for category, subdimensions in self.dimensions.items():
            phenomenology[category] = {}
            for subdim in subdimensions:
                key = f"{category}.{subdim}"
                mapper = self.mappers[key]

                features = self._extract_features(functional_props)
                prediction = mapper.predict(features)
                phenomenology[category][subdim] = float(prediction)

        return phenomenology

    def predict_with_uncertainty(self, functional_props: dict) -> tuple:
        """Predict phenomenology + uncertainty bounds."""
        # Use ensemble or Bayesian approach for uncertainty
        predictions = []

        # Bootstrap predictions
        for _ in range(100):
            # Resample training data
            bootstrap_data = np.random.choice(self.training_data,
                                             size=len(self.training_data))
            # Train on bootstrap sample
            temp_mapper = PhenomenologyMapper()
            temp_mapper.training_data = list(bootstrap_data)
            temp_mapper.train()
            # Predict
            pred = temp_mapper.predict(functional_props)
            predictions.append(pred)

        # Compute mean and std
        mean_pred = self._average_phenomenology(predictions)
        std_pred = self._std_phenomenology(predictions)

        return mean_pred, std_pred

    def interpret_profile(self, phenomenology: dict) -> str:
        """Generate natural language interpretation of phenomenological profile."""
        interpretations = []

        # Perception
        perc = phenomenology["perception"]
        if perc["clarity"] < 0.3:
            interpretations.append("dim, unclear perceptual experience")
        elif perc["clarity"] > 0.8:
            interpretations.append("crystal-clear, vivid perception")

        if perc["richness"] > 0.8:
            interpretations.append("overwhelming perceptual richness")
        elif perc["richness"] < 0.3:
            interpretations.append("sparse, minimal perceptual content")

        # Affect
        affect = phenomenology["affect"]
        if affect["intensity"] < 0.2:
            interpretations.append("emotionally flat")
        elif affect["intensity"] > 0.8:
            interpretations.append("intense emotional experience")

        if affect["valence"] > 0.7:
            interpretations.append("predominantly positive affect")
        elif affect["valence"] < 0.3:
            interpretations.append("predominantly negative affect")

        # Unity
        unity = phenomenology["unity"]
        if unity["integration"] > 0.8 and unity["boundaries"] < 0.3:
            interpretations.append("profound sense of unity, dissolved boundaries")
        elif unity["integration"] < 0.3:
            interpretations.append("fragmented, disintegrated experience")

        # Volition
        volition = phenomenology["volition"]
        if volition["agency"] < 0.2:
            interpretations.append("minimal sense of agency, passive experience")
        elif volition["agency"] > 0.8:
            interpretations.append("strong sense of will and control")

        # Temporality
        time = phenomenology["temporality"]
        if time["nowness"] > 0.8 and time["flow"] < 0.3:
            interpretations.append("timeless, eternal present")
        elif time["flow"] > 0.8:
            interpretations.append("rapid temporal flow, racing thoughts")

        return "; ".join(interpretations)

    def compare_to_human_baseline(self, phenomenology: dict) -> dict:
        """Compare predicted phenomenology to normal waking baseline."""
        baseline = self.predict(NORMAL_WAKING_FUNCTIONAL)

        differences = {}
        for category in phenomenology:
            differences[category] = {}
            for subdim in phenomenology[category]:
                diff = phenomenology[category][subdim] - baseline[category][subdim]
                differences[category][subdim] = diff

        return differences

    def _extract_features(self, functional_props: dict) -> np.ndarray:
        """Convert functional properties dict to feature vector."""
        # Include interactions and nonlinear transforms
        features = []

        # Direct features
        features.extend([
            functional_props.get("k", 0),
            functional_props.get("k_meta", 0),
            functional_props.get("b", 0),
            functional_props.get("Φ", 0),
            functional_props.get("W", 0),
            functional_props.get("A", 0),
            functional_props.get("R", 0)
        ])

        # Interactions
        features.append(functional_props.get("k", 0) * functional_props.get("b", 0))  # k*b
        features.append(functional_props.get("k_meta", 0) * functional_props.get("R", 0))  # k_meta*R
        features.append(functional_props.get("Φ", 0) * functional_props.get("W", 0))  # Φ*W

        # Nonlinear transforms
        features.append(functional_props.get("k", 0) ** 2)
        features.append(np.log(functional_props.get("Φ", 0) + 0.01))

        return np.array(features)

    def _fit_mapper(self, X, y, weights):
        """Fit regression model (can use linear, polynomial, or neural network)."""
        from sklearn.ensemble import RandomForestRegressor
        from sklearn.preprocessing import StandardScaler

        # Standardize features
        scaler = StandardScaler()
        X_scaled = scaler.fit_transform(X)

        # Fit model
        model = RandomForestRegressor(n_estimators=100, max_depth=5)
        model.fit(X_scaled, y, sample_weight=weights)

        return {"scaler": scaler, "model": model}
```

## Limitations and Caveats

### Fundamental Limitations

**L1: The Hard Problem Remains Unsolved**
We're mapping functional→reported phenomenology, not explaining WHY phenomenology exists.

**L2: Report Unreliability**
Phenomenological reports are:
- Memory-dependent (distorted)
- Language-limited (ineffable experiences)
- Subject to interpretation biases

**L3: Inaccessible States**
Some consciousness states leave no memory (deep sleep, some anesthesia, death). We can measure functional properties but get no phenomenological reports.

**L4: Other Minds Problem**
Even with perfect mapping for humans, we cannot verify it generalizes to AI. We're making principled inferences, not certain knowledge.

**L5: Substrate Dependence Unknown**
We don't know which phenomenological dimensions are substrate-independent vs implementation-specific.

### Methodological Limitations

**M1: Sample Bias**
Reports come from:
- WEIRD populations (Western, Educated, Industrialized, Rich, Democratic)
- Meditators (experienced introspectors)
- Psychedelic users (self-selected for openness)

**M2: Temporal Resolution**
Functional measurements (EEG, fMRI) have ~10-100ms resolution. Phenomenology may have finer temporal structure.

**M3: State Dependence**
Phenomenological reports depend on current state during reporting, not just during experience.

**M4: Causal Ambiguity**
Correlation between functional and phenomenological doesn't prove causation. Both might be caused by third factor.
