#### 2. Experimental Designs (4 types)

**Randomized Controlled Trial (RCT)** - Gold standard:
```python
CONTROL:   No intervention → measure k
TREATMENT: Apply intervention → measure k
CAUSAL EFFECT = k_treatment - k_control
```

**Factorial (2×2)** - Test interactions:
```python
        | No B      | B
--------|-----------|----------
No A    | Control   | Only B
A       | Only A    | A + B

Tests: Main effect A, Main effect B, Interaction (A × B)
```

**Dose-Response** - Find relationship:
```python
DOSES: [0.0, 0.25, 0.5, 0.75, 1.0]
FIT: k = a + b*dose (linear? threshold? saturation?)
```

**Within-Subject** - Before/after:
```python
BASELINE: Measure k before
POST: Apply intervention → measure k
EFFECT = k_post - k_baseline
```

#### 3. Causal Analysis

```python
analysis = {

}
```
