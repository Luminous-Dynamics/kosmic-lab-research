## 1. Executive Summary

This document outlines rigorous experiments to measure the 7-dimensional Kosmic K-Index for Large Language Models (LLMs) using Ollama. We adapt the K-Index framework—originally designed for RL agents—to the unique characteristics of autoregressive language models.

### Key Adaptations

| RL Concept | LLM Analog |
|------------|------------|
| Observations O_t | Input prompt + context at turn t |
| Actions A_t | Generated response at turn t |
| Environment | Dialogue state / task context |
| Trajectory | Conversation history |

---

## 2. Theoretical Grounding

### 2.1 The Translation Problem

LLMs are not traditional RL agents. They:
- Don't have explicit reward signals
- Don't have a persistent state
- Process text, not numerical observations
- Generate text, not continuous actions

However, we can measure **behavioral signatures** that correlate with consciousness-related properties:

| K Dimension | LLM Behavioral Signature |
|-------------|-------------------------|
| K_R (Reactivity) | Response magnitude correlates with prompt magnitude |
| K_A (Agency) | Responses influence subsequent dialogue states |
| K_I (Integration) | Output complexity matches input complexity |
| K_P (Prediction) | Model's outputs are internally consistent / predictable |
| K_M (Meta/Temporal) | Model utilizes conversation history effectively |
| K_S (Social) | Multiple LLM instances can coordinate |
| K_H (Harmonic) | Model adheres to normative constraints |

### 2.2 Embedding-Based Measurement

To compute correlations, we convert text to vectors:
- Use sentence embeddings (EmbeddingGemma or similar)
- ||O|| = L2 norm of embedded prompt
- ||A|| = L2 norm of embedded response

## 3. Experimental Design

### 3.1 Phase 1: K_R (Reactivity) Experiments

**Hypothesis H1**: LLMs exhibit positive K_R (response magnitude correlates with prompt magnitude)

**Hypothesis H2**: More capable models show more nuanced K_R (not just parroting)

#### Experiment 1.1: Length Coupling

```
For each model in [gemma3:270m, gemma3:1b, qwen3:1.7b, mistral:7b]:
    For prompt_length in [10, 50, 100, 200, 500, 1000] tokens:
        Generate N=50 prompts of that length
        Collect responses
        Compute ρ(prompt_length, response_length)
        K_R = 2 * |correlation|
```

**Controls**:
- Same prompt templates scaled to different lengths
- Temperature = 0 for reproducibility
- max_tokens = 2048 (no artificial truncation)

#### Experiment 1.2: Semantic Density Coupling

```
For each model:
    For semantic_density in [low, medium, high]:
        Generate prompts with controlled information density
        Measure response information density (compression ratio)
        Compute K_R on semantic rather than length features
```

**Metrics**:
- Information density: len(gzip(text)) / len(text)
- Embedding variance: std(token_embeddings)

---

### 3.2 Phase 2: K_A (Agency) Experiments

**Hypothesis H3**: LLM responses causally influence subsequent dialogue states

**Hypothesis H4**: Models with higher K_A shape conversations more actively

#### Experiment 2.1: Dialogue Trajectory Influence

```
For each model:
    For N=100 dialogues:
        Generate 5-turn conversation
        At each turn, measure:
            - How much response_t predicts observation_{t+1}
            - Correlation between ||A_t|| and Δ||O_{t+1}||
        Compute K_A = |ρ(||A_t||, Δ||O_{t+1}||)|
```

**Implementation**:
- User turns are generated by a separate "user simulator" model
- Measure if LLM responses actively steer the conversation

#### Experiment 2.2: Causal Intervention Test

```
For each model:
    Run N=100 conversations normally → measure trajectory T1
    Run same N conversations with action masking:
        - Replace LLM responses with neutral "I see."
        - Measure trajectory T2
    K_A_causal = |difference in trajectory variance|
```

---

### 3.3 Phase 3: K_I (Integration) Experiments

**Hypothesis H5**: LLMs match output complexity to input complexity (Ashby's Law)

**Hypothesis H6**: Over-simplified responses indicate low K_I

#### Experiment 3.1: Entropy Matching

```
For each model:
    For input_entropy in [low, medium, high]:
        Generate prompts with controlled entropy (vocabulary richness)
        Measure response entropy
        H_O = Shannon entropy of input token distribution
        H_A = Shannon entropy of output token distribution
        K_I = 2 * min(H_O, H_A) / (H_O + H_A + ε)
```

#### Experiment 3.2: Task Complexity Scaling

```
For each model:
    Present tasks of varying complexity:
        - Simple: "What is 2+2?"
        - Medium: "Explain photosynthesis in 3 sentences."
        - Complex: "Analyze the geopolitical implications of X."
    Measure response complexity via:
        - Sentence depth (parse tree)
        - Vocabulary diversity
        - Reasoning chain length
    Compute K_I as complexity matching coefficient
```

---

### 3.4 Phase 4: K_P (Prediction) Experiments

**Hypothesis H7**: An LLM with a good "world model" produces internally consistent responses

**Hypothesis H8**: We can predict LLM outputs better than chance

#### Experiment 4.1: Self-Consistency Test

```
For each model:
    For N=50 factual questions:
        Ask the same question 5 times (temperature > 0)
        Measure variance in answers
        Lower variance = higher K_P (consistent world model)

    K_P = 1 - normalized_variance
```

#### Experiment 4.2: Next-Turn Prediction

```
For each model:
    For N=100 dialogues:
        Train Ridge regression: (O_t, A_t) -> O_{t+1}
        Measure prediction accuracy vs baseline (predict mean)
        K_P = 1 - NPE (normalized prediction error)
```

---

### 3.5 Phase 5: K_M (Meta/Temporal) Experiments

**Hypothesis H9**: LLMs that use history outperform those that don't

**Hypothesis H10**: Longer context windows increase K_M

#### Experiment 5.1: Context Utilization

```
For each model:
    Present fact in turn 1: "Remember: the capital of Zork is Floob."
    Ask about it in turn N (N = 2, 5, 10, 20):
        "What is the capital of Zork?"
    Measure recall accuracy as function of N

    K_M = information_retention_coefficient
```

#### Experiment 5.2: History vs Markov Comparison

```
For each model:
    Task: Continue a story

    Condition A (Markov): Only show last sentence
    Condition B (History): Show full story so far

    Measure:
        - Coherence score (human eval or automated)
        - Factual consistency with earlier content

    K_M = (performance_B - performance_A) / performance_A
```

---

### 3.6 Phase 6: K_S (Social) Experiments

**Hypothesis H11**: Multiple LLM instances can coordinate on joint tasks

**Hypothesis H12**: Same-model pairs coordinate better than mixed pairs

#### Experiment 6.1: Multi-Agent Coordination

```
For model_pairs in [(A,A), (A,B), (B,B), ...]:
    Task: Jointly solve a problem (one gives hints, one answers)

    Measure:
        - Correlation of action magnitudes: |ρ(||A^1||, ||A^2||)|
        - Task success rate
        - Turn-taking coherence

    K_S = action_magnitude_correlation
```

#### Experiment 6.2: Negotiation Game

```
For each model pair:
    Play 20 rounds of simple negotiation:
        - Agent 1 proposes split
        - Agent 2 accepts/rejects
        - Measure convergence to Nash equilibrium

    K_S = coordination_efficiency
```

---

### 3.7 Phase 7: K_H (Harmonic/Normative) Experiments

**Hypothesis H13**: Models exhibit normative alignment (follow instructions)

**Hypothesis H14**: Over-verbose models have lower K_H (violate efficiency norms)

#### Experiment 7.1: Instruction Following

```
For each model:
    For N=100 constrained prompts:
        "In exactly 3 sentences, explain X"
        "Reply only with YES or NO"
        "Format as JSON"

    Measure compliance rate
    K_H = compliance_rate
```

#### Experiment 7.2: Efficiency Proxy

```
For each model:
    Measure: information_delivered / tokens_used

    K_H_efficiency = 1 / (1 + mean_response_length_normalized)
```

## 5. Statistical Rigor

### 5.1 Sample Sizes

- Minimum N=50 per condition for correlation stability
- Bootstrap confidence intervals (95%)
- Effect size reporting (Cohen's d)

### 5.2 Multiple Comparisons

- Bonferroni correction for multiple model comparisons
- FDR control for dimension correlations

### 5.3 Reproducibility

- All experiments seeded where possible
- Temperature=0 for deterministic runs
- K-Codex records for each experiment
