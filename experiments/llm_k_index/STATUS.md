## What the New Improvements Add

### #11.8: Uncertainty Quantification

**Why It's Revolutionary**:
- Every scientific estimate needs confidence intervals
- Distinguishes epistemic (reducible) from aleatoric (irreducible) uncertainty
- Tells us WHEN to trust estimates vs collect more data

**Example Usage**:
```python
# Compute consciousness with uncertainty
profile = ConsciousnessProfile(model, data, use_bma=True)
uncertainty = quantify_uncertainty(
    profile.theory_scores,
    profile.theory_weights
)

print(f"K: {uncertainty.mean:.3f} ± {uncertainty.std:.3f}")
print(f"95% CI: [{uncertainty.confidence_interval_95[0]:.3f}, "
      f"{uncertainty.confidence_interval_95[1]:.3f}]")
print(f"Confidence: {uncertainty.confidence:.3f}")
print(f"Epistemic: {uncertainty.epistemic_uncertainty:.3f}")
print(f"Aleatoric: {uncertainty.aleatoric_uncertainty:.3f}")
```

**Real Impact**: When theories disagree (high epistemic), we know to collect more evidence!

### #11.9: Counterfactual Reasoning

**Why It's Revolutionary**:
- Moves from correlation to CAUSATION
- Answers "what if" questions using Pearl's do-calculus
- Identifies critical components

**Example Usage**:
```python
# What if IIT was perfect?
reasoner = CounterfactualReasoner(causal_dag, bma_weights)
intervention = Intervention('IIT', InterventionType.SET, value=1.0)
result = reasoner.compute_counterfactual(theory_scores, intervention)

print(f"Effect of perfect IIT: {result.total_effect:+.3f}")
print(f"Affected theories: {result.affected_theories}")

# Which theory is most critical?
critical = reasoner.find_critical_theories(theory_scores)
for theory, effect in critical:
    print(f"{theory}: removing drops K by {abs(effect):.3f}")
```

**Real Impact**: Can test interventions (training, modifications) BEFORE implementing them!
