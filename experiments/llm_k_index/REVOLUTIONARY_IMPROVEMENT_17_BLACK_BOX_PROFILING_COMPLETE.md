## 📏 Proxy Measurement Methods

### Bayesian Uncertainty Quantification

For each dimension, we estimate consciousness from behavioral data with **explicit uncertainty bounds**:

```python
@dataclass
class ProxyMeasurement:
    dimension: str
    proxy_estimate: float  # Best estimate from behavioral data
    confidence_interval: Tuple[float, float]  # Uncertainty bounds (95% CI)
    measurement_type: str  # "proxy" vs "direct"
    evidence: List[ProbeResult]  # Supporting behavioral evidence
    uncertainty_sources: List[str]  # Known limitations

    # Validation data (if available)
    ground_truth: Optional[float] = None  # Direct measurement
    proxy_error: Optional[float] = None  # |proxy - ground_truth|
```

### Proxy Method Example: IIT φ

```python
def estimate_iit_phi(probe_results: List[ProbeResult]) -> ProxyMeasurement:
    """
    Estimate IIT φ from behavioral integration tests.

    METHOD:
    1. Apply integration probes (multi-element dependencies)
    2. Score responses for integration quality
    3. Aggregate scores with uncertainty quantification

    VALIDATION: Tested against systems with known φ values
    UNCERTAINTY: ±0.15 (95% CI) based on validation studies
    """
    integration_results = [r for r in probe_results
                          if r.probe.probe_type == ProbeType.INTEGRATION]

    if not integration_results:
        return ProxyMeasurement(
            dimension="IIT_phi",
            proxy_estimate=0.5,  # Default
            confidence_interval=(0.0, 1.0),  # Max uncertainty
            measurement_type="proxy",
            evidence=[],
            uncertainty_sources=["No integration probes applied"]
        )

    # Aggregate scores with Bayesian updating
    scores = [r.score for r in integration_results]
    weights = [r.confidence for r in integration_results]

    # Weighted mean
    estimate = weighted_average(scores, weights)

    # Uncertainty from variance + measurement noise
    variance = weighted_variance(scores, weights, estimate)
    std_error = math.sqrt(variance + 0.15**2)  # Add measurement uncertainty

    # 95% confidence interval
    ci = (
        max(0.0, estimate - 1.96 * std_error),
        min(1.0, estimate + 1.96 * std_error)
    )

    return ProxyMeasurement(
        dimension="IIT_phi",
        proxy_estimate=estimate,
        confidence_interval=ci,
        measurement_type="proxy",
        evidence=integration_results,
        uncertainty_sources=[
            "Behavioral proxy (no direct neural access)",
            "Limited probe coverage",
            "NLP scoring imperfect"
        ]
    )
```

def weighted_average(values, weights):
    total_weight = sum(weights)
    return sum(v * w for v, w in zip(values, weights)) / total_weight

def weighted_variance(values, weights, mean):
    total_weight = sum(weights)
