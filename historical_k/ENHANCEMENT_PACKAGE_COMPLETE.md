## 💡 Pro Tips for Integration

**Tip 1**: Start with easiest additions (Abstract, Conclusion) to build momentum
**Tip 2**: Compile after EACH major addition to catch LaTeX errors immediately
**Tip 3**: Use `\label{}` and `\ref{}` consistently for cross-references
**Tip 4**: Copy table code exactly (whitespace matters in LaTeX tables)
**Tip 5**: Keep backup copy of original manuscript before starting

**Common LaTeX Issues**:
- Missing `\usepackage{array}` for table `p{width}` columns → add to preamble
- Table too wide → use `\small` or `\footnotesize` font
- Missing cross-reference → run `pdflatex` 3 times (not just once)
- Special characters (%, &, $) → escape with backslash (\\%, \\&, \\$)
